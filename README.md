# Leaderboard Bot

Discord bot to manage and publish leaderboards for sim-racing challenges

![sample bot answer](https://i.imgur.com/soq1C5y.png)

## setup

install dependencies with `npm install`

create `.env` with

```
APPLICATION_ID=
BOT_TOKEN=
DB_URI=
```

## run

`npm start` or `npm run watch`

### dev tools

vscode
mongodb

extensions:

- prettier
